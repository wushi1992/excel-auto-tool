# excel-auto-tool
# CSV 自动化数据分析工具
基于 Python 实现的 CSV 文件批量处理、数据分析与自动报表生成工具，支持文件夹 / ZIP 压缩包读取，自动生成带图表的可视化 Excel 汇总报告。

## 项目简介
该工具用于批量处理大量 CSV 数据文件，自动完成：
- 批量读取文件夹 / ZIP 中的 CSV
- 自动数据清洗、异常值过滤
- 线性回归计算（t197 / t195 / trtime / trlt / ltcd）
- 自动生成散点图
- 输出带图表的专业 Excel 汇总报告
全程无需手动操作，一键运行，大幅提升数据处理效率。

## 技术栈
- 语言：Python 3
- 数据处理：Pandas, NumPy
- 科学计算：Scipy
- 图表可视化：Matplotlib
- Excel 生成：OpenPyXL
- 文件处理：OS, Zipfile, IO

## 核心功能
1. 支持批量读取**文件夹**或 **ZIP 压缩包**内的 CSV 文件
2. 自动跳过无效数据、空值、格式异常文件
3. 自动执行**线性回归分析**
4. 自动生成**数据散点图**
5. 自动生成**带图表的 Excel 汇总报告**
6. 完整异常处理，稳定不崩溃

## 使用方法
1. 修改代码中的路径：
   ROOT_FOLDER = "你的CSV文件所在文件夹或ZIP路径"
2. 安装依赖：
   pip install pandas numpy matplotlib scipy openpyxl
3. 运行脚本：
   python main.py
4. 自动在桌面生成：csv处理结果汇总_带图表.xlsx
